# SimpleInterpreter
一个简易的脚本语言解释器
## 一、特点
### 语句的分隔符
与C、Java等语言不同，该脚本多条语句之间的分隔方式更类似于Python。即通过换行符表示语句的结束，并且通过语句前'\t'的个数判断语句的结构。
### 二、变量的数据类型
1.数字
定义一个数字变量i并设置初始值为0
i=0
2.字符串
定义一个字符串变量i并设置初始值为"abc"
i="abc"
